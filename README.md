# JAVA-Tips
## Math method in Java 
```
Math.sqrt(16)    平方根
Math.cbrt(8)     立方根
Math.pow(a,b)    计算a的b次方
Math.max(,)   
Math.min(,)
Math.abs()
Math.ceil()      返回取整最大值 11.5 取到 12，所有小数均向上取整
Math.floor()     返回取整最小值，11.5取到11， 所有小数向下取整
Math.random()    取一个大于等于零，小于1的随机数
Math.rint()      四舍五入 但是.5的时候优先返回偶数 比如10.5取到10 11.5取到12
Math.round()     四舍五入，float 返回int， double返回long
```
